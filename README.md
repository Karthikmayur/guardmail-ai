# GuardMail AI — Phishing Email Detector

AI-powered phishing detection running locally on your machine.
Built with Python, Flask, and Claude AI.

## Setup
1. pip install flask anthropic flask-cors
2. Create .env file: ANTHROPIC_API_KEY=your-key-here
3. python app.py
4. Open http://localhost:5000
